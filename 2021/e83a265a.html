CSharp 配置和创建新项目

发表于2021-12-31|更新于2023-12-30|编程语言C-Sharp
|总字数:546|阅读时长:1分钟|浏览量:

在 https://visualstudio.microsoft.com/zh-hans/ 下载 Visual Studio

下载 Community 2019（其他两个版本属于专业版，需要付费）社区版的功能齐全，所以下载社区版就行。

安装之后需要等待，因为微软服务器在国外，所以要安装很久。

为了方便打开 Visual Studio，

以 Win 10 为例，打开开始菜单，在里面搜 Visual Studio Installer 和 Visual Studio 2019，可以右键将其固定到开始菜单：

先打开 Visual Studio Installer，点击 Visual Studio Community 2019 的 Modify

在弹出的窗口中，勾选 .NET desktop development，然后点击右下角的 Install while downloading，等待下载。

然后我们打开 Visual Studio，首先会叫你注册账号，下面有个不显眼的链接可以点击直接跳过（应该是叫 Skip）

点击 Create New Project, 在上方搜 Console，然后在列表中选择 Console App（注意每个列表项中有语言，选择 C# 的），

点击右下角的 Next，然后我们可以给项目名称 Project Name 取上自己喜欢的名字，其他的目录默认就行（可以选择自己喜欢的位置）。最后点击 Next，选择 .Net Core 3.1，再点击 Create 创建项目。

最后 Visual Studio 界面长这样：

Unicode 乱码解决

有时候运行会出现乱码，如下

打开设置，搜索 region 或者 地区，打开 Region Setting / 地区设置。

把 Country or region 改为 China

滑倒最下方，找到 Additional date, time & regional settings。如果是全屏设置的话，那么这个选项就在右侧

点开 Region（区域选项）

点击 Administrative 标签页，点击 Change system locale… 按钮

在弹出框里的下拉菜单找到 Chinese (Simplified, China)，点击 Ok，点击 Apply，然后关闭设置界面，可能需要重启才能生效。

代码提示

我们可以通过 ctrl + space 组合键在任意字中（字符串除外）唤出智能代码提示。

如果没有弹出任何代码提示，肯定是 ctrl + space 组合键被其他软件占用了。通常是被微软输入法占用了。这时候右键微软输入法，然后点击 Key configuration

在弹出的界面中取消勾选 Ctrl + Space 即可
